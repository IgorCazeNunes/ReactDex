## 📦️ Como utilizar o projeto

```bash
  # Clone o repositório
  ❯ git clone https://github.com/IgorCazeNunes/ReactDex.git

  # Entre no diretório
  ❯ cd reactdex/mobile

  # Instale as dependências
  ❯ yarn
  # ou pode usar o `npm`
  ❯ npm install
  
  # Inicialize o projeto
  ❯ yarn android
  # ou pode usar o `npm`
  ❯ npm run android
```
